# Permis de Conduire - Questions d'Examen

Application mobile pour s'entraîner aux questions de l'examen pratique du permis de conduire français.

## Fonctionnalités

- **VI - Vérification Intérieure**: Questions sur les commandes et équipements à l'intérieur du véhicule
- **VE - Vérification Extérieure**: Questions sur les éléments extérieurs du véhicule (pneus, niveaux, etc.)
- **QSER - Questions de Sécurité Routière**: Questions sur le code de la route et la sécurité
- **1ers Secours**: Questions sur les gestes de premiers secours en cas d'accident

## Démarrage

1. Installer les dépendances

   ```bash
   npm install
   ```

2. Lancer l'application

   ```bash
   npx expo start
   ```

## Technologies

- [Expo](https://expo.dev) - Framework React Native
- [Expo Router](https://docs.expo.dev/router/introduction) - Navigation basée sur les fichiers
- TypeScript

## Déploiement

L'application est déployée sur [Vercel](https://vercel.com).
